⚙️ Firebase設定が必要です

下記の手順でFirebaseを設定してください。設定後にこのメッセージは消えます。

① console.firebase.google.com でプロジェクトを新規作成

② 「Firestore Database」を有効化 → ルールを以下に設定：

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /prices/{doc} {
      allow read: if true;
      allow create: if true;
      allow update: if request.resource.data.diff(resource.data).affectedKeys()
                       .hasOnly(['likes']);
      allow delete: if false;
    }
  }
}

③ プロジェクト設定 →「ウェブアプリを追加」→ 設定値をこのHTMLの下記の箇所に貼り付け：

const firebaseConfig = {
  apiKey: "YOUR_API_KEY",       // ← ここを置き換え
  authDomain: "...",
  projectId: "...",
  ...
};
